CarShop.dataなどが入っている場所にtest.txtを入れてください
内容は空でも構いません

GUIの見方
それぞれボタンを押すことで反応します

1.押さないでボタン
ルールにCarRule16が追加されます
デバック用です

2.doボタン
追加された文(ADDなんとか)、ルール一覧
また別ウィンドウで過程を表示します
この別ウィンドウですがdoを押すたび複製される欠陥品なので
あまり連打はしない方がいいです

3.deleteボタン
ボタンを入力後Eclipseの入力にてルール名を
入力するとそのルールを削除します
入力例　　"CarRule15"  注意:"マークも入力してください
そのルールがない場合無視されます

4.addボタン
deleteと同じくEclipseで入力します
入力例　　"CarRule16"
次 ifの中身を入力します
入力例　　"?x is a car" ifはいらないですが"は必要です
次 ifの中身を続けて書くか決めます
入力例　　yes 上記に戻る
        no 下記へ進む
次　thenの中身を入力します
入力例　　"?x is a Toyota"

注意　入力に"がないまま書き込むとバグるのでバグったら
直接データ内を書き換えてください

また何のバグがあるかわからないのでCarShop.dataの控えはどこかに残しておいて下さい
